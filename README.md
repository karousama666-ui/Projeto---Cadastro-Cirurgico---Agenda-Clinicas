# Projeto---Cadastro-Cirurgico---Agenda-Clinicas
Desenvolvimento de sistema de cadastro de agenda cirúrgica para clinicas.


# Sistema de Agenda Cirúrgica desenvolvido em Python.

Funcionalidades:
- Cadastro de cirurgias
- Consulta por paciente
- Consulta por data
- Edição de registros
- Exclusão de registros
- Persistência de dados em JSON
- Validação de datas e horários
- Relatórios resumidos por hospital
